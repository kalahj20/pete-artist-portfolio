# Pete Thinkful | Artist Portfolio

A clean, responsive personal portfolio website for Pete, a Denver-based artist specializing in abstract and graffiti art.

Overview

This project is a single-page web portfolio designed to showcase Pete's background as an artist, featured works, and professional contact information. It uses semantic HTML5 to ensure accessibility and a structured layout.

Features

* **Navigation:** Internal anchor links for "About," "Portfolio," and "Contact" for a seamless user experience.
* **About Section:** Professional bio, background, and artistic interests.
* **Porfolio Section:** A dedicated portfolio section featuring key works like *Abstract Red* and *Melted Rainbow*.
* **Contact Information:** Direct links to LinkedIn, Instagram, and Pinterest.

Built With:

* **HTML5:** Semantic structure.
* **CSS3:** Custom styling (via `style.css`).
* **Normalize.css:** For cross-browser style consistency.

Project Structure
* **index.html**          (Main portfolio page)

* **style.css**          (Custom stylesheet)
