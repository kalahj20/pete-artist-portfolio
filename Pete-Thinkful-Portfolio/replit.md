# Pete Thinkful - Static Site

## Overview
A static HTML/CSS portfolio site for Pete Thinkful.

## Structure
- `index.html` - Main page
- `style.css` - Stylesheet
- `images/` - Image assets
- `.config/static-web-server.toml` - Web server config (port 5000)

## Running
The "Serve static" workflow runs `static-web-server` on port 5000.
